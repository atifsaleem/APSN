== Installation ==

1. Upload 'my-live-chat-for-wp' directory to Wordpress plugins directory ('/wp-content/plugins/')
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the 'MyLiveChat' menu on the left.
4. Follow the instructions.

== Frequently Asked Questions ==

= What makes My LiveChat different from other live chat companies? =

Live chat systems can become very complex and difficult to use. We've engineered My LiveChat so that anyone can use it, from the first time they log into the system. My LiveChat is powered by Cute Chat, the #1 web chat software. It can support hundreds of web chat rooms and thousands of visitors without any degradation in response time. It's still being actively developed.

= What is the minimum contract period? =

There is no minimum contract period. You can cancel the service at any time.

= Can you do proactive chats and site monitoring with My Live Chat?  =

Yes, you can monitor visitors on your website in real-time. You can also send "proactive" chat invitations to those visitors.

= I have multiple websites I would like to support. Is this possible with My LiveChat? =

Yes, with My LiveChat's visitor tracking software, you will know which URL they are coming from, before you start any chat sessions.

= Can I host My LiveChat on my own servers? =

No. My LiveChat is a fully hosted, subscription based software service. My Live Chat can take care of all server, network and code update issues seamlessly. This allows your organization more time to spend on your core competencies. If you want to a live chat software hosted in your own server, please check <a href="http://cutesoft.net/live-support/">Cute Live Support</a>.
